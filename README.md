# 起きる · Okiru

Rastreador de animes pessoal. Guarda sua lista, notas e te dá badges conforme você assiste.

---

### O que tem

- Lista com status (assistindo, completo, na lista…)
- Nota de 0 a 10 por anime
- Badges automáticas baseadas no seu histórico
- Compartilhar como story ou post

### Como usar

Abra o `index.html` direto no navegador. Nenhuma instalação necessária.

---

Feito pra uso próprio. Sem backend, sem conta, tudo local.
